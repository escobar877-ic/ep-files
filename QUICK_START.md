# ⚡ Быстрый старт EP-Files v2.0

## 🚀 Запуск за 3 шага

### 1. Установите зависимости

```bash
# Backend
pip install -r requirements.txt

# На macOS (для python-magic):
brew install libmagic

# На Ubuntu/Debian:
sudo apt-get install libmagic1

# Frontend
cd frontend
npm install
cd ..
```

### 2. Примените миграции

```bash
source venv/bin/activate  # или venv\Scripts\activate на Windows
python manage.py migrate
```

### 3. Запустите проект

```bash
# Автоматический запуск (рекомендуется)
./run.sh

# ИЛИ вручную:
# Terminal 1:
python manage.py runserver

# Terminal 2:
cd frontend && npm run dev
```

- **Backend**: http://localhost:8000

## 🎮 Использование

### Регистрация
1. Откройте http://localhost:5173
2. Нажмите "Регистрация"
3. Заполните форму (имя, email, пароль)
4. Войдите в систему

### Загрузка файлов
1. Перейдите на страницу "Мои файлы"
2. Нажмите "Загрузить файл"
3. Выберите файл
4. Наблюдайте за прогресс-баром! 📊

### Статус-бар хранилища
На странице "Мои файлы" вы увидите:

## 🔒 Безопасность

### Что защищено

✅ Блокировка опасных файлов (.exe, .bat, .sh и т.д.)
✅ Проверка реального типа файла (не только расширения)
✅ Только владелец может скачивать/удалять свои файлы
✅ Rate limiting (100 запросов/час для анонимов)
✅ Логирование всех действий
✅ Security headers (XSS, Clickjacking защита)

### Попробуйте загрузить

- ✅ PDF, изображение, архив - **разрешено**
- ❌ .exe, .bat, .sh файлы - **заблокировано**
- ❌ Файлы > 100 MB - **заблокировано**

## 📊 API Endpoints

### Основные
```
POST /api/auth/register/     - Регистрация
POST /api/auth/login/        - Вход
GET  /api/files/             - Список файлов
POST /api/upload/            - Загрузка файла
GET  /api/download/<id>/     - Скачивание
DELETE /api/files/<id>/      - Удаление
GET  /api/storage/stats/     - Статистика хранилища
GET  /api/search/?q=<query>  - Поиск файлов
```

## 🐛 Решение проблем

### python-magic не устанавливается

```bash
# macOS
brew install libmagic

# Ubuntu/Debian
sudo apt-get install libmagic1

# Windows
pip install python-magic-bin
```

### Ошибка миграции

```bash
# Удалите базу (ВНИМАНИЕ: потеряете данные!)
rm db.sqlite3

# Создайте заново
python manage.py migrate
```

### Frontend не запускается

```bash
cd frontend
rm -rf node_modules package-lock.json
npm install
npm run dev
```
