# EP Files

Веб-приложение для безопасной загрузки, хранения и совместного доступа к файлам. Backend на Django, frontend на React/Vue. Поддержка прав доступа, публичных ссылок, предпросмотра и админки.

## 🚀 Как начать работу

**Перед созданием ветки: прочитайте [CONTRIBUTING.md](CONTRIBUTING.md)**  
- Git Flow, имена веток (`feature/задача`), коммиты (`feat: ...`), MR в `dev`.

## Быстрый старт
- `docker-compose up` (после настройки)
- Backend: `uvicorn main:app --reload`
- Frontend: `npm run dev`
