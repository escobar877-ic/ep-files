## Запуск проекта:

### 1. Backend (Django):

```bash
# Активируйте виртуальное окружение
source venv/bin/activate  # для macOS/Linux
# или
venv\Scripts\activate  # для Windows

# Примените миграции
python manage.py migrate

# Запустите сервер
python manage.py runserver
```

Backend будет доступен на `http://localhost:8000`

### 2. Frontend (React):

```bash
# Перейдите в папку frontend
cd frontend

# Установите зависимости (если еще не установлены)
npm install

# Запустите dev-сервер
npm run dev
```

Frontend будет доступен на `http://localhost:5173`

## Использование:

1. **Регистрация**: Перейдите на `/register` и создайте аккаунт
2. **Вход**: Войдите с вашими данными на `/login`
3. **Загрузка файлов**: 
   - Нажмите кнопку "Загрузить файл"
   - Выберите файл
   - Наблюдайте за прогресс-баром загрузки
4. **Скачивание**: Нажмите иконку скачивания рядом с файлом
5. **Удаление**: Нажмите иконку корзины для удаления файла

## Структура API:

### Аутентификация:
- `POST /api/auth/register/` - регистрация (name, email, password)
- `POST /api/auth/login/` - вход (email, password)
- `GET /api/auth/me/` - информация о текущем пользователе

### Файлы:
- `GET /api/files/` - список файлов пользователя
- `POST /api/upload/` - загрузка файла (multipart/form-data)
- `GET /api/download/<id>/` - скачивание файла
- `DELETE /api/files/<id>/` - удаление файла

## Примечания:

- Файлы сохраняются в папку `storage/files/`
- Максимальный размер файла: 100 MB (настраивается в settings.py)
- CORS настроен для localhost:5173, localhost:3000
