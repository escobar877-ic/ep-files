EP-Files Documentation
======================

EP-Files — школьный веб-сервис для хранения, обмена, просмотра и редактирования файлов.
Документация описывает основные модули backend-части проекта, модели данных,
API-представления, сервисы, права доступа, middleware и вспомогательные функции.
