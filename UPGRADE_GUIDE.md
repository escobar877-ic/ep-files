## Обновление проекта

### Шаг 1: Обновление зависимостей

```bash
# Backend
pip install python-magic

# Если на macOS:
brew install libmagic

# Если на Ubuntu/Debian:
sudo apt-get install libmagic1
```

### Шаг 2: Применение миграций

```bash
source venv/bin/activate
python manage.py migrate
```

### Шаг 3: Создание папки для логов

```bash
mkdir -p logs
```

### Шаг 4: Обновление настроек

Проверьте `main/settings.py`:
- Добавлены новые middleware
- Настроено логирование
- Улучшены настройки безопасности

### Шаг 5: Перезапуск серверов

```bash
# Остановите текущие процессы (Ctrl+C)

# Запустите заново
./run.sh
```

## Миграция данных

### Существующие файлы

Все существующие файлы останутся без изменений. Новые проверки применяются только к новым загрузкам.

### Существующие пользователи

Все пользователи сохранят доступ к своим файлам.

## Проверка после обновления

### 1. Проверка загрузки файлов

```bash
# Попробуйте загрузить:
- Обычный файл (PDF, изображение) - должно работать
- Исполняемый файл (.exe) - должно блокироваться
- Большой файл (>100MB) - должно блокироваться
```

### 2. Проверка статус-бара

- Откройте страницу /files
- Проверьте отображение статистики
- Загрузите файл - статистика должна обновиться

### 3. Проверка поиска

- Введите имя файла в поиск
- Нажмите "Найти"
- Проверьте результаты

### 4. Проверка безопасности

```bash
# Проверка Django
python manage.py check --deploy

# Должны быть предупреждения только о DEBUG=True
```

## Откат изменений

Если что-то пошло не так:

### 1. Откат миграций

```bash
python manage.py migrate ep_files_app 0001_initial
```

### 2. Восстановление файлов

```bash
git checkout HEAD~1 ep_files_app/api/views.py
git checkout HEAD~1 ep_files_app/permissions.py
git checkout HEAD~1 ep_files_app/validators.py
```

### 3. Перезапуск

```bash
python manage.py runserver
```

## Известные проблемы

### 1. python-magic не устанавливается

**Решение:**
```bash
# macOS
brew install libmagic
pip install python-magic

# Ubuntu/Debian
sudo apt-get install libmagic1
pip install python-magic

# Windows
pip install python-magic-bin
```

### 2. Ошибка при миграции

**Решение:**
```bash
# Удалите базу данных (ВНИМАНИЕ: потеряете данные!)
rm db.sqlite3

# Создайте заново
python manage.py migrate
python manage.py createsuperuser
```

### 3. Статус-бар не отображается

**Решение:**
- Проверьте консоль браузера на ошибки
- Убедитесь, что backend запущен
- Проверьте, что endpoint /api/storage/stats/ работает

## Поддержка

При возникновении проблем:

1. Проверьте логи: `logs/app.log`
2. Проверьте консоль браузера (F12)
3. Проверьте консоль Django сервера
